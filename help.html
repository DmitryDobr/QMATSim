QMatSim

About

This is a small plugin that provides simple converter from vector geometry to MatSim Network .xml file.

MatSim project page: https://matsim.org/


Input Data

Plugin needs a point vector layer, where every point is unique node with unique id. And plugin needs a line vector layer, where every line represents link between two nodes.


Point Data

Every point need an unique ID represented by attribute or just use point numder at layer by default.


Line Data

Every line need an unique ID represented by attribute or just use point numder at layer by default. Also need an attribute to determine the direction of movement on the line. One-sided or two-sided. Also layer need to have attributes:

    • freespeed - maximum speed on a line-section of road;
      capacity - maximum number of cars that can be on the road;
      permlanes - the number of traffic lanes on the line-section of the road;
